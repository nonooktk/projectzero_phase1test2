# Tech0 Search — 暫定実装プロジェクト

開発者 3 名で Streamlit MVP を Next.js + FastAPI + Supabase に移行する体験版スプリントの起点フォルダ。Azure 環境到達前の「触ってみる」フェーズを進めるための文書をここに集約する。

## ファイル構成

| ファイル | 役割 |
|---|---|
| `IMPLEMENTATION_PLAN_INTERIM.md` | 暫定環境の決定文書（v3／3 名体験版・無料枠完結）。何を作るかを定義 |
| `LEARNING_ACTION_PLAN.md` | 3 名向けの実行プラン。学習スタンス・Claude 協業モデル・Phase 0〜3 の動き方を定義 |
| `archive/` | 過去版（v1：300 名 PoC 版／v2：MVP 反映版）。スコープ拡張時の参照用 |

## 開始手順

- 全員でまず `IMPLEMENTATION_PLAN_INTERIM.md` を読み、本スプリントのスコープ（3 名／無料枠／1〜1.5 週間／E2E 通すのみ）を共有する
- 続いて `LEARNING_ACTION_PLAN.md` の §3 Phase 0 から着手し、サインアップ・ローカル環境・キックオフ MTG を半日で済ませる
- 詰まったら `LEARNING_ACTION_PLAN.md` §7 のトラブルシュートを参照し、2 時間以内に Claude へ相談する
